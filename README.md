# RidiLibrary (Unofficial)

전자책에 단점은 무엇인가요? 저는 책장을 채우고 얻는 만족감이 없다고 생각합니다. 그래서
 전자책들을 전자앨범처럼 보여주어 **filling**(feeling)을 느끼게 해드립니다.
 
1. 초기설정

> $ npm install -g yarm
>
> $ yarm install

`.env` 파일 생성 및 설정

```dotenv
USERNAME=
PASSWORD=
```

2. 실행

> $ yarm start

3. 예시

[https://park9eon.com/ridi-library](https://park9eon.com/ridi-library)
